## Features per Theme

### Velocity
- **Stack**: Tailwind CDN + Google Fonts (Inter, Open Sans)
- **Brand**: "ClubHub"
- **Palette**: Amber/Slate
- **Style**: Rounded cards, blur navbar, pill buttons

### Heritage
- **Stack**: Bootstrap 5 + Google Fonts (Playfair Display, Lato)
- **Brand**: "The Society"
- **Palette**: Navy (#1a2744) / Gold (#D4A520)
- **Style**: Gold borders, dark footer, elegant badges

### Terra
- **Stack**: Pure CSS + Google Fonts (Montserrat)
- **Brand**: "GreenOrg"
- **Palette**: Forest (#2D5016) / Earth (#6B3410) / Accent (#E85D04 "sunset road")
- **Style**: Diagonal hero clip-path, racing stripe cards, hover micro-animations

### Zen
- **Stack**: Pure CSS + System fonts
- **Brand**: "Mindful"
- **Palette**: Ink (#0A0A0A) / Paper (#FFFFFF) + Pop colors (Magenta, Yellow, Blue, Green)
- **Style**: Comic/Manga panels, halftone patterns, speed lines, speech bubbles, color pop

### Clubs
- **Stack**: Tailwind CDN + System fonts
- **Brand**: "EagleRiders"
- **Palette**: Black (#0A0A0A) / Red (#C41E3A)
- **Style**: Full-height hero, uppercase headings, border cards, Italian motorsport aesthetic

### Tricolore ğŸ‡®ğŸ‡¹
- **Stack**: Pure CSS + Google Fonts (Montserrat)
- **Brand**: "MotoItalia"
- **Palette**: Green (#009246) / White (#F4F5F0) / Red (#CE2B37) / Gold (#C9A227)
- **Style**: Tricolore ribbons, decorative cockades, gradient badges, hover translateX
- **Target**: Motorcycle tourists in love with Italy
- **Content**: Iconic itineraries (Stelvio, Amalfi, Chianti, Etna, Motor Valley)
